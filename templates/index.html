Welcome to Meminisse!

This is a personal project web app dedicated entirely for my own usage, but you can use it too! or even help me make this web app even more useful, consider dropping to my Github page and fork it. And now to the details! first things first this web app is designed to be as light as possible, and will not save any data that you enter on this site in any way.

So what is the purpose of this website? Well the true purpose is to help me create a simple, easy to remember, but secure passwords generating an individualized based sequence that is based to your personal preference. Because complex password is just a bit hard to remember making us jotting it down somewhere to avoid forgetting it, Meminisse aims to remove this entirely by using words or sentence that is closely related to you and only you, and generating the password based on that.
